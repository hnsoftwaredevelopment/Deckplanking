# Modelboat Deckplanking - Guía De Usuario

## Objetivo

Modelboat Deckplanking ayuda a calcular y visualizar el entablado de cubierta para modelos de barcos. La aplicación calcula la longitud de las tablas, la longitud de los segmentos, los patrones de desplazamiento y muestra una vista previa escalable de la cubierta.

## Datos Del Proyecto

Introduce la escala, el ancho de la cubierta, el ancho de la tabla, la longitud de la tabla y el patrón de desplazamiento deseado. Puedes usar un número fijo de filas o dejar que la aplicación lo calcule a partir del ancho de la cubierta y el ancho de la tabla.

Activa la tabla central si la cubierta la utiliza. En ese caso también puedes introducir el ancho de esa tabla.

## Vista Previa

La vista previa muestra la cubierta con segmentos de tablas, regla, dirección de proa y popa, tabla central opcional y clavos de cubierta. Usa los botones de zoom para acercar o alejar. La sección de entrada se puede contraer para dejar más espacio a la vista previa.

Usa los ajustes de contorno para aproximar el estrechamiento de proa y popa. La forma está pensada como una aproximación práctica, no como un plano completo con escotillas, orificios de mástil u otras aberturas.

## Proyectos

Usa los botones de proyecto para crear, abrir, guardar, renombrar o eliminar un proyecto. Los archivos de proyecto usan la extensión `.deckplanking.json`.

## Exportar E Imprimir

Puedes exportar la vista previa como PNG o PDF. En Windows también puedes imprimir directamente. En Android la aplicación usa la pantalla estándar del dispositivo para compartir o guardar.

## Ajustes

En Ajustes puedes elegir idioma, tema y sistema de medida. La aplicación admite visualización métrica e imperial. Los ajustes se guardan localmente.

## Feedback

Usa Feedback para informar de un error o solicitar una función. Para errores, la aplicación puede incluir información técnica como plataforma, versión del sistema operativo, modelo del dispositivo y versión de la aplicación.
