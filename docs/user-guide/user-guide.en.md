# Modelboat Deckplanking - User Guide

## Purpose

Modelboat Deckplanking helps you calculate and visualize deck planking for model ships. The app calculates plank length, segment length, butt-shift patterns, and shows a scalable deck preview.

## Project Data

Enter the scale, deck width, plank width, plank length, and preferred shift pattern. You can use a fixed row count or let the app calculate the row count from deck width and plank width.

Enable the king plank when the deck uses a central plank. When enabled, you can also enter the king plank width.

## Preview

The preview shows the deck with plank segments, ruler, bow and stern direction, optional king plank, and trenails. Use the zoom buttons to zoom in and out. The input section can be collapsed to give the preview more room.

Use the contour settings to approximate bow and stern tapering. The shape is intended as a practical approximation, not as a complete construction drawing with hatches, mast holes, or other deck openings.

## Projects

Use the project buttons to create, open, save, rename, or delete a project. Project files use the `.deckplanking.json` extension.

## Export And Print

You can export the preview as PNG or PDF. On Windows you can also print directly. On Android the app uses the device's standard share or save screen.

## Settings

In Settings you can choose language, theme, and measurement system. The app supports metric and imperial display. Settings are stored locally.

## Feedback

Use Feedback to report a bug or request a feature. For bugs, the app can include technical information such as platform, OS version, device model, and app version.
